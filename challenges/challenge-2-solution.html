The buttons have been deleted. Create an array with four colours in it. Loop through the array and create four buttons and append them to the colour-box. Give the buttons the id of the colour and the class of the colour. You will need to change the changeColourBox function to change the style rather than class as the classes have been deleted from the CSS.

Hints:

  • use <element>.style.backgroundColor to set the colour without using a class
  • you can create elements with document.createElement
  • you can use appendChild to add the new buttons
  • use any 4 web colour names rather than a hex as it will also be a label for the button but make sure they are contrasting
